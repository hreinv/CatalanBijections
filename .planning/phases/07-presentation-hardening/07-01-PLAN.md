<task type="auto">
  <name>Task 1: Add keyboard shortcuts and tab-switch resilience to main.js</name>
  <files>js/main.js</files>
  <action>
Add two event listeners inside the DOMContentLoaded block in main.js, after the resize handler:

1. **Keyboard handler (`document.addEventListener('keydown', onKeyDown)`):**
   - Define a SPEED_PRESETS constant: `{ '1': 0.5, '2': 1.0, '3': 1.5, '4': 2.0, '5': 3.0 }`
   - Define `onKeyDown(e)` function:
     - Guard: if `e.target.tagName` is `'SELECT'`, `'INPUT'`, or `'TEXTAREA'`, return immediately (let native behavior work)
     - `' '` (space): `e.preventDefault()`, call `engine.togglePlay()`, call `updatePlayPauseButton()`
     - `'ArrowRight'`: `e.preventDefault()`, call `engine.stepForward()`
     - `'ArrowLeft'`: `e.preventDefault()`, call `engine.stepBackward()`
     - `'Home'`: `e.preventDefault()`, call `engine.jumpToStart()`
     - `'End'`: `e.preventDefault()`, call `engine.jumpToEnd()`
     - Default: check if `SPEED_PRESETS[e.key]` is defined. If so, call `engine.setSpeed(speed)`, set `dom.speedSlider.value = speed`, set `dom.speedDisplay.textContent = speed.toFixed(1) + 'x'`. If not, return without calling preventDefault.
   - Use `e.preventDefault()` ONLY for handled keys (Space, ArrowRight, ArrowLeft, Home, End). Do NOT preventDefault for unhandled keys or number keys (they don't need it).

2. **Visibility change handler (`document.addEventListener('visibilitychange', ...)`):**
   - If `document.hidden` is true AND `engine` is not null AND `engine.isPlaying()` is true:
     - Call `engine.pause()`
     - Call `updatePlayPauseButton()`
   - Do NOT auto-resume on visibility return (presenter should explicitly resume with spacebar)

Place the SPEED_PRESETS constant at module scope near the top of the file (after the state object, before the rendering context variables). Place the onKeyDown function definition near the other event handler functions (after onNext, before the animation engine section). Wire both listeners inside DOMContentLoaded after the existing resize handler.
