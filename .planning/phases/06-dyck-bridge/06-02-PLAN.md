- When there are active steps AND the path has more than 2 nodes (composed bijection):

Replace it: if result is null and source !== target, this is unexpected -- show "Loading bijection..." as a transient state, or simply remove the branch. **4. Update `resetAnimation()`:**
- Also reset `state.animation.path = []` alongside the existing resets.
